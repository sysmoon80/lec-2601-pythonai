4w-todo05.py — 1단~9단 가로 출력 (중첩 for문)

코드 전체 구조 (2단계)

┌─────────────────────────────────────────────────────┐
│  STEP 1: 단 제목 출력 (단일 for문)                     │
│  STEP 2: 구구단 본체 출력 (중첩 for문)                  │
└─────────────────────────────────────────────────────┘

STEP 1: 단 제목 출력 (Line 23~25)

for i in range(1, 10):
    print(f"  # {i}단 # ", end="")
print()

핵심 개념: end=""는 줄바꿈을 하지 않고 같은 줄에 이어서 출력한다.

반복 흐름:
┌──────┬──────────────────────┬────────────────┐
│  i   │  출력 내용            │  줄바꿈?       │
├──────┼──────────────────────┼────────────────┤
│  1   │  "  # 1단 # "        │  ❌ (end="")   │
│  2   │  "  # 2단 # "        │  ❌ (end="")   │
│  3   │  "  # 3단 # "        │  ❌ (end="")   │
│ ...  │  ...                 │  ❌            │
│  9   │  "  # 9단 # "        │  ❌ (end="")   │
└──────┴──────────────────────┴────────────────┘
         ↓ for문 끝난 후
      print()  ← 여기서 줄바꿈! ✅

화면 결과:
  # 1단 #   # 2단 #   # 3단 #  ...  # 9단 #
──────────────────────────────────────────────  ← 줄바꿈

STEP 2: 구구단 본체 출력 (Line 28~31) — ⭐ 핵심

for i in range(1, 10):        # 바깥 루프: 곱할 수 (행)
    for dan in range(1, 10):  # 안쪽 루프: 단 (열)
        print(f"{dan:2d} x{i:2d}={dan * i:2d} ", end="")
    print()                   # 한 행 끝나면 줄바꿈

변수 역할 이해

변수 역할 범위 비유
i 곱할 수 (행 번호) 1→9 세로로 내려가는 줄
dan 단 (열 번호) 1→9 가로로 나열되는 단

바깥이 1칸 움직일 때마다, 안쪽은 9칸 전부 돈다!
총 실행 횟수: 9 × 9 = 81번

포맷 문자열 해석

f"{dan:2d} x{i:2d}={dan*i:2d} "
  │  │      │        │
  │  │      │        └── 결과값을 2자리로 (정렬용)
  │  │      └── 곱할 수를 2자리로
  │  └── 2d = 최소 2자리, 정수(decimal)
  └── 단

예: dan=2, i=3 → " 2 x 3= 6 "
예: dan=9, i=9 → " 9 x 9=81 "
                       ↑↑
                  :2d 덕분에 한 칸이면 앞에 공백 추가 → 열이 깔끔하게 정렬됨

학생들이 헷갈리는 3가지 포인트

# 헷갈리는 것 정리
1 i가 단이 아니라 곱할 수? 가로 출력이라 i(바깥)=행=곱할 수, dan(안쪽)=열=단. 세로 출력이면 반대!
2 end=""가 뭐 하는 거? print()는 기본적으로 끝에 줄바꿈(\n)을 추가. end=""로 바꾸면 줄바꿈 없이 이어서 출력
3 print()가 왜 빈 채로? 안쪽 for문이 한 행을 다 출력한 후, 다음 행으로 넘어가기 위한 줄바꿈 전용
