## 1. VS Code란 무엇인가?

### 1.1 VS Code 소개

**Visual Studio Code**는 마이크로소프트에서 개발한 무료 오픈소스 코드 에디터입니다.

- **가볍고 강력**: 필요한 기능만 탑재하여 가볍지만 강력한 기능 제공
- **다양한 언어 지원**: Python, JavaScript, C++, Java 등 400개 이상의 프로그래밍 언어 지원
- **풍부한 확장**: 확장 기능으로 기능 추가 가능
- **크로스 플랫폼**: Windows, macOS, Linux 모두 지원

### 1.2 왜 VS Code인가?

| 기능 | 장점 |
|------|------|
| **통합 터미널** | 에디터 내에서 바로 터미널 사용 |
| **디버깅** | 코드 실행 중 오류 추적 및 디버깅 |
| **Git 통합** | 버전 관리 바로 확인 및 커밋 |
| **IntelliSense** | 코드 자동 완성 및 제안 |
| **확장 마켓플레이스** | 필요한 도구를 확장으로 설치 |

## 2. VS Code 설치

### 🎯 Step 1: VS Code 다운로드

> **실습**: 웹 브라우저에서 VS Code를 다운로드하세요.

**다운로드 페이지**: https://code.visualstudio.com/

### 🎯 Step 2: 운영체제별 설치

#### macOS

```bash
# Homebrew를 사용한 설치 (터미널)
brew install --cask visual-studio-code
```

또는 웹에서 다운로드 후:
1. 다운로드 폴더에서 `Visual Studio Code.dmg` 더블 클릭
2. VS Code 아이콘을 **Applications** 폴더로 드래그
3. Launchpad에서 VS Code 실행

#### Windows

**PowerShell에서 Winget 사용**:
```powershell
winget install Microsoft.VisualStudioCode
```

또는 웹에서 다운로드 후:
1. `VSCodeUserSetup-x64.exe` 다운로드
2. 설치 파일 더블 클릭
3. "다음" → "동의함" → "설치" 클릭
4. 완료 후 "재시작" 또는 "VS Code 실행" 클릭

### ✅ 확인: 설치 확인

#### macOS

```bash
# 터미널에서 버전 확인
code --version
```

**예상 출력**:
```
1.96.4
a622c65b4c0c5a1b85cf8a8b5675512a04e9a902
arm64
```

#### Windows

```powershell
# PowerShell에서 버전 확인
code --version
```

### 🎯 Step 3: VS Code 실행

- **macOS**: Command + Space → "VS Code" 입력
- **Windows**: Win + S → "VS Code" 입력

## 7. 실습 과제

### 과제 1: 간단한 계산기 만들기

**목표**: VS Code에서 간단한 계산기를 만들어보세요.

1. `calculator.py` 파일 생성
2. 다음 코드 작성:

```python
# calculator.py
def add(a, b):
    return a + b

def subtract(a, b):
    return a - b

def multiply(a, b):
    return a * b

def divide(a, b):
    if b == 0:
        return "0으로 나눌 수 없습니다"
    return a / b

def main():
    print("=== 간단한 계산기 ===")
    print("1: 덧셈, 2: 뺄셈, 3: 곱셈, 4: 나눗셈")

    choice = input("선택: ")
    num1 = float(input("첫 번째 숫자: "))
    num2 = float(input("두 번째 숫자: "))

    if choice == "1":
        print(f"결과: {add(num1, num2)}")
    elif choice == "2":
        print(f"결과: {subtract(num1, num2)}")
    elif choice == "3":
        print(f"결과: {multiply(num1, num2)}")
    elif choice == "4":
        print(f"결과: {divide(num1, num2)}")
    else:
        print("잘못된 선택입니다")

if __name__ == "__main__":
    main()
```

3. **F5**로 실행
4. 각 기능 테스트

### 과제 2: 디버깅 실습

**목표**: 디버거를 사용하여 코드 실행 과정을 확인하세요.

1. `debug_practice.py` 파일 생성
2. 다음 코드 작성:

```python
# debug_practice.py
def factorial(n):
    """팩토리얼 계산"""
    if n <= 1:
        return 1
    return n * factorial(n - 1)

def fibonacci(n):
    """피보나치 수열"""
    if n <= 1:
        return n
    return fibonacci(n - 1) + fibonacci(n - 2)

def main():
    numbers = [5, 6, 7]

    for num in numbers:
        fact = factorial(num)
        fib = fibonacci(num)
        print(f"{num}! = {fact}, fib({num}) = {fib}")

if __name__ == "__main__":
    main()
```

3. 중단점 설정:
   - 8번 라인 (`return n * factorial(n - 1)`)
   - 24번 라인 (`fact = factorial(num)`)
4. **F5**로 디버깅 시작
5. 변수 값 확인하며 한 단계씩 실행

## 9. 문제 해결

### 문제 1: Python 인터프리터를 찾을 수 없음

**증상**: "No Python interpreter found"

**해결 방법**:

1. UV로 Python 설치 확인:
```bash
uv python install 3.12
```

2. 가상환경 생성:
```bash
uv venv
```

3. VS Code에서 인터프리터 다시 선택:
   - **Command/Ctrl + Shift + P**
   - "Python: Select Interpreter"
   - `.venv/bin/python` 선택

### 문제 2: 확장 설치 실패

**증상**: 확장 설치 중 오류 발생

**해결 방법**:

1. VS Code 재시작
2. 인터넷 연결 확인
3. VS Code 업데이트: **Command/Ctrl + Shift + P** → "Update"

### 문제 3: 단축키가 작동하지 않음

**증상**: 단축키 입력 시 반응 없음

**해결 방법**:

1. 단축키 설정 확인:
   - **Command/Ctrl + K, Command/Ctrl + S**
   - 충돌하는 단축키 확인

2. 키보드 레이아웃 확인:
   - macOS: 시스템 환경설정 → 키보드 → 입력 소스
