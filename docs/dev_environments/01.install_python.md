## 1. UV란 무엇인가?

### 1.1 UV 소개

**UV**는 Rust로 작성된 차세대 Python 패키지 관리자로, 다음과 같은 특징을 가집니다:

- **초고속**: 기존 pip보다 10-100배 빠름
- **올인원**: pip, virtualenv, pyenv, pip-tools를 하나로 대체
- **Python 버전 관리**: 여러 Python 버전을 쉽게 설치하고 관리

### 1.2 UV vs 기존 도구

| 구분 | 기존 방식 | UV 방식 |
|------|----------|---------|
| 패키지 설치 | `pip install` | `uv add` 또는 `uv pip install` |
| 가상환경 | `python -m venv` | `uv venv` |
| 버전 관리 | `pyenv install` | `uv python install` |
| 속도 | 느림 | 매우 빠름 |

## 2. UV 설치

### 🎯 Step 1: UV 설치

> **실습**: 터미널을 열고 UV를 설치해보세요.

#### macOS

```bash
# curl을 사용한 설치
curl -LsSf https://astral.sh/uv/install.sh | sh
```

#### Windows (PowerShell)

```powershell
# PowerShell installer 사용
irm https://astral.sh/uv/install.ps1 | iex
```

### ✅ 확인: 설치 확인

```bash
# UV 버전 확인
uv --version
```

**예상 출력**:
```
uv 0.5.12
```

## 3. Python 버전 관리

### 🎯 Step 2: 사용 가능한 Python 버전 확인

```bash
# 사용 가능한 Python 버전 목록
uv python list
```

**예상 출력**:
```
3.12.1
3.11.8
3.10.14
...
```

### 🎯 Step 3: Python 설치

```bash
# Python 3.12 설치
uv python install 3.12 --default
```

**설치 진행 과정**:
```
Installed Python 3.12.1 to /Users/username/.local/share/uv/python/cpython-3.12.1
```

### 🎯 Step 4: 기본 Python 버전 설정

### ✅ 확인: Python 버전 확인

```bash
# Python 버전 확인
python --version
# 또는
python3 --version
```

**예상 출력**:
```
Python 3.12.1
```
