## 문제 10. f-string 출력

다음 코드의 실행 결과는?

```python

eng = 85
total = kor + eng
print(f"합계: {total}점, 평균: {total/2:.1f}점")
```

1. `합계: 175점, 평균: 87.5점`
2. `합계: 175점, 평균: 87점`
3. `합계: 90 85점, 평균: 87.5점`
4. `합계: {total}점, 평균: {total/2:.1f}점`

---

## 정답

1. **2번** - 변수는 데이터의 위치를 가리키는 이름표와 같은 역할을 한다
2. **3번** - `score = 90` (파이썬은 등호 한 개(=)를 사용하여 변수에 값을 할당한다)
3. **3번** - `_score` (밑줄로 시작할 수 있음)
4. **1번** - `1` (나머지 연산자 %는 7을 3으로 나눈 나머지를 반환)
5. **2번** - `26` (a=10 → a+=3으로 a=13 → a*=2로 a=26)
6. **3번** - `a == b`는 `False` (10은 5와 같지 않음)
7. **2번** - `False` (a<b는 거짓, and는 두 조건이 모두 참일 때만 참)
8. **2번** - `3` (3**2=9 먼저 계산, 4*2=8, 그 다음 2+9-8=3)
9. **3번** - `str` (문자열)
10. **1번** - f-string을 사용하여 변수값과 포맷팅이 적용된 출력
