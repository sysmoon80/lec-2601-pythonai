## 문제 5. if-elif-else문

다음 코드의 실행 결과는?

```python
score = 85
if score >= 90:
    grade = 'A'
elif score >= 80:
    grade = 'B'
elif score >= 70:
    grade = 'C'
else:
    grade = 'F'
print(grade)
```

1. A
2. B
3. C
4. F

## 문제 10. 중첩 if문 vs if-elif-else

중첩 if문과 if-elif-else문에 대한 설명으로 올바른 것은?

1. 중첩 if문이 항상 if-elif-else문보다 성능이 좋다
2. if-elif-else문이 항상 중첩 if문보다 성능이 좋다
3. 중첩 if문은 가독성이 높고, if-elif-else문은 프로그램 실행시간을 단축시킨다
4. 중첩 if문은 프로그램 실행시간을 단축시키고, if-elif-else문은 가독성이 높다

## 정답

1. **2번** - 조건문은 특정 조건에 따라 실행의 흐름을 다르게 하는 제어문이다
2. **3번** - `if 조건식: 코드` (파이썬은 콜론(:)을 사용하여 조건식과 코드를 구분한다)
3. **3번** - 파이썬에서 들여쓰기는 문법의 일부로, 잘못된 들여쓰기는 에러를 발생시킨다
4. **2번** - `불합격` (score=65는 70 미만이므로 else 블록 실행)
5. **2번** - `B` (score=85는 90 미만이지만 80 이상이므로 elif 블록 실행)
6. **1번** - `True` (a>8은 참, b>3은 참, and는 두 조건이 모두 참일 때 참)
7. **2번** - `False` (age=15는 7 이하도 아니고 65 이상도 아니므로, or는 둘 중 하나라도 참이어야 참)
8. **2번** - `False` (x>5는 참, not은 참을 거짓으로 바꿈)
9. **2번** - `if height >= 120 and age >= 10:` (두 조건이 모두 만족해야 하므로 and 사용)
10. **4번** - 중첩 if문은 프로그램 실행시간을 단축시키고, if-elif-else문은 가독성이 높다
