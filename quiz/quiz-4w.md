## 문제 9. continue 문

다음 코드의 실행 결과는?

```python
total = 0
for i in range(5):
    if i % 2 == 0:
        continue
    total += i
print(total)
```

1. 6
2. 10
3. 4
4. 0

---

## 문제 10. 다중 for문

다음 코드가 출력하는 별 패턴은?

```python
for i in range(3):
    for j in range(5):
        print("*", end="")
    print()
```

1. 3행 5열의 사각형
2. 5행 3열의 사각형
3. 왼쪽 직각삼각형
4. 피라미드

## 정답

1. **2번** - for문은 정해진 횟수만큼 반복하거나 순차자료를 순회하며 반복하는 명령어다
2. **2번** - `1 3 5` (range(1, 6, 2)는 1부터 6 미만까지 2씩 증가)
3. **3번** - 0부터 시작하여 4까지 1씩 증가 (기본값: 시작=0, 증가=1)
4. **1번** - `hello` (문자열을 한 문자씩 순회하며 출력)
5. **3번** - `300` (70+80+90+60 = 300)
6. **2번** - `25` (1+3+5+7+9 = 25)
7. **2번** - 왼쪽 직각삼각형 (*, **, *** 형태로 출력)
8. **2번** - 반복문을 즉시 종료하고 빠져나간다
9. **3번** - `4` (range(5)는 0,1,2,3,4이고, 짝수(0,2,4)는 continue로 건너뛰고 홀수(1,3)만 더함: 1+3=4)

10. **1번** - 3행 5열의 사각형 (바깥 for문이 3번, 안쪽 for문이 5번 실행)
